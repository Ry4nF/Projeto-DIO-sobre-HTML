Tags de Escrita

Para a escrita de títulos pode ser usado tags de "h1" a "h6" (Tamanho de letra maior à menor).

A tag "blockquote" pode ser utilizaa para uma observação ou citação, centralizando a frase contida em tal tag.

A tag "p" pode ser usada para a definição de um parágraf no texto

Para modificar a formatação de palavras pode ser usadas variadas tags, como tag "strong", que serve para deixar o texto em negrito.

Outra tag é a "i" para a formatação em itálico

Para que as palavras fiquem e formato sublinhado pode ser usado a tag "u"

Para uma marcação no texto, pode se usar a tag "mark"

Alem dessas existe uma grande diversidade de tags, podendo ser encontradas em sites na internet, como: https://pt.semrush.com/blog/tags-html/

Imagens

Em HTML, as imagens são incorporadas usando a tag "img". Essa tag não tem uma tag de fechamento, já que não envolve conteúdo, apenas referencia um recurso de imagem. A tag "img" tem um atributo src (obrigatório) que indica o caminho (URL ou caminho relativo) para a imagem. Além disso, atravéz da tag "width" é possivel moldar o tamanho da imagem

Como colocar links

Para adicionar links em páginas HTML, utilize a tag "a" com o atributo href para indicar o URL de destino. Por exemplo: https://www.exemplo.com O texto entre as tags e será exibido como o link na página, redirecionando para o endereço especificado no atributo href quando clicado.

Target

A tag target é usada em conjunto com a tag "a" (âncora/link) para especificar onde o navegador deve abrir o destino do link quando clicado. Ela define o contexto de exibição do recurso vinculado. Os valores mais comuns para o atributo são o "_self"(Abre o link na mesma janela ou guia) e o "_blank"(abre o link em uma janela/guia diferente).

Title

A tag title é usada para fornecer um texto de dica de ferramenta (tooltip) quando o cursor do mouse passa sobre o link. Essa dica de ferramenta exibe informações adicionais sobre o link, oferecendo contexto ou descrição adicional ao usuário.

Input

Os elementos "input" em HTML são usados para coletar informações dos usuários. Eles podem assumir diferentes tipos, como texto, botões, caixas de seleção, entre outros.
